# Pasta de Imagens

Coloque aqui todas as imagens e ícones da landing page.

## Arquivos necessários:

1. **foto-perfil.jpg** - Sua foto para a seção "Sobre Mim"
   - Tamanho recomendado: 400x500px ou proporção similar
   - Formato: JPG, PNG ou WEBP

2. **favicon.ico** - Logo para aparecer na aba do navegador
   - Tamanho recomendado: 32x32px ou 64x64px
   - Formato: ICO, PNG ou SVG
   - Pode também usar: favicon.png, favicon.svg

3. Outras imagens que você queira adicionar à landing page

## Observações:

- Os nomes dos arquivos são case-sensitive (maiúsculas/minúsculas importam)
- Para trocar o formato da imagem (ex: foto-perfil.png), atualize o HTML também
